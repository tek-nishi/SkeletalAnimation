# Assimp Loader Sample with Cinder 0.8.6
Assimpで読み込んだモデルデータ(アニメーション含む)をCinder 0.8.6で表示するサンプルです。

自分の学習向けなので内容は雑です。

以下のサンプルを含んでいます。

+ Assimpでモデルデータを読み込む
+ メッシュデータ
+ マテリアル
+ 階層アニメーション
+ スキンアニメーション
+ iOS対応


## How to build
Cinderはもとより、Assimpのインクルードファイルとライブラリファイルをプロジェクトから参照できるようにしておいてください。
